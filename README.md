# Sitting-Ducks
A game developed by Matthew Merritt and Michael Merritt for the Quinnipiac Game Dev Club eJam 2020.

The jam page with results can be found [here](https://itch.io/jam/quinnipiac-game-dev-club-ejam-2020), and the original jam submission (as a standalone Windows executable) can be found [here](https://meepix.itch.io/sitting-ducks).
